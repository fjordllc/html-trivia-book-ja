# 付録B HTML人物列伝

## Tim Berners-Lee（ティム・バーナーズ＝リー）

CERN（欧州原子核研究機構）で情報共有の仕組みとして Web を提案し、URL・HTTP・HTML を含む最初の土台を整えた人物です。本書の文脈で重要なのは、単に HTML を発明したことではありません。文書、場所、通信を公開された組として設計し、Web を特定製品の中ではなく、分散した文書の網として始めたことです。

## Ian Hickson（イアン・ヒクソン）

WHATWG の HTML 仕様編集で中心的な役割を果たした人物です。本書との関係で言えば、現実のブラウザ挙動を細かく仕様へ落とし込む作業を強く推し進めた存在として重要です。HTML5 から Living Standard へ至る流れの中で、理想だけでなく実装の一致を仕様に書き込む方向へ大きな影響を与えました。

## Håkon Wium Lie（ホーコン・ウィウム・リー）

CSS の提案者として知られる人物です。HTML 単体では担えない見た目の制御を外へ切り出す発想を形にし、HTML と CSS の役割分担を決定づけました。本書では主役ではありませんが、presentational な要素が増えていった HTML に対して、意味と見た目を分ける方向を現実の仕組みにした点で非常に重要です。
